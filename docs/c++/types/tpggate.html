TpgGate¶

class TpgGate : public TpgObjBase¶

ゲートの情報を表すクラス

主な目的は縮退故障/遷移故障などのゲート構造に基づいた故障の 設定用の情報を提供すること

外部入力/外部出力も「ゲート」とみなす．

TpgFault stem_fault(Fval2 fval) const¶

ステムの故障を返す．

  • 故障タイプが網羅故障/遷移故障でない場合には不正な呼び出しとなる．

  • 定義されていない場合は空の故障を返す．

パラメータ:

fval -- [in] 故障値
