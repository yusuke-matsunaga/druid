TpgFFR¶

class TpgFFR : public TpgObjBase¶

TpgNetwork の FFR の情報を表すクラス

具体的には以下の情報を持つ．

  • FFR の根のノード

  • FFR の葉のノードのリスト

  • FFR に含まれる全ノードのリスト

  • ffR に含まれる故障のリスト 一度設定された不変のオブジェクトとなる．
