TpgMFFC¶

class TpgMFFC : public TpgObjBase¶

TpgNetwork の MFFC(Maximal Fanout Free Cone) の情報を表すクラス

具体的には以下の情報を持つ．

  • MFFC の根のノード

  • MFFC に含まれる FFR のリスト

  • MFFC に含まれる故障のリスト 一度設定されたら不変のオブジェクトとなる．
